A [Flarum](http://flarum.org) extension. extend money extension

## Installation

Install with composer:

```sh
composer require cobbz/flarum-ext-money-batch-transfer:"*"
```

## Updating

```sh
composer update cobbz/flarum-ext-money-batch-transfer:"*"
php flarum migrate
php flarum cache:clear
```
